Network Traffic Classification

  • Implemented a 3-layer neural network classifier using PyTorch with ReLU activation functions and dropout regularization to prevent overfitting

Rust-Haskell Language Interoperability

  • Built a DSL interpreter demonstrating cross-language interoperability between Rust and Haskell using Foreign Function Interface (FFI)
  • Implemented typechecking and type inference in Haskell to validate expressions with overloaded functions for integer and vector types
  • Designed low-level evaluation logic in Rust, with all data structures created and managed through the Rust memory model for better performance
  • Used FFI to bridge the languages: Haskell performs high-level type analysis while Rust handels efficient runtime execution and memory management
  • Implemented unsafe Rust operations with raw pointers and external C functions to enable seamless data passing between the two languages
